Class spriteOverlay

A class which is used to draw sprites relative to others.

Inherits from sprite. Could be used to overlay clothing on a sprite or a rotating arm, for example.

See also:

    sprite

Methods

spriteOverlay:new (spriteInstOrArgs) Creates a spriteOverlay.
spriteOverlay:attach (sprite) Attaches this spriteOverlay onto the given sprite.
spriteOverlay:detach () Detached this spriteOverlay from the sprite it's currently attached to.
spriteOverlay:draw () Draws this spriteOverlay using the same drawing behavior as sprite._draw, but uses values relative to the parent.
x, y, and rotation are additive (self value + parent value),
w and h are multiplicative (self value * parent value),
flipHorizontal and flipVertical are XOR'd (self value ~= parent value),
ox and oy are multiplied by the spriteOverlay's width/height after multiplication (self value * w/h).
