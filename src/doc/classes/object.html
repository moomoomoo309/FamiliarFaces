Class object

The "object" class, so to speak.

Adds callbacks.

Methods

object:addCallback (key, fct) Adds a callback to the given property (running the function when the property changes)
object:triggerCallback (property, value) Triggers the callback for the given property, updating it to the passed value if not cancelled.
object:defaultMetatable () Returns the default metatable for this object.
object:new (tbl) Creates a new object using the values contained in tbl.

object:removeGlobalCallback () Removes the global callback from the object.

Methods

object:addCallback (key, fct)
Adds a callback to the given property (running the function when the property changes)

Parameters:

  • key any The property to add the callback for.
  • fct function The function to run when key changes. Object and the new value will be passed as parameters to it.

Returns:

    nil
object:triggerCallback (property, value)
Triggers the callback for the given property, updating it to the passed value if not cancelled.

Parameters:

  • property string The property to trigger callbacks for.
  • value any What the new value of the property should be.

Returns:

    nil
object:defaultMetatable ()
Returns the default metatable for this object.

Returns:

    table The default metatable for this object.
object:new (tbl)
Creates a new object using the values contained in tbl. Can be run using object(), object.new() or object:new().

Parameters:

  • tbl table The parameters to the object. All of these values will be put into the object on initialization.

Returns:

    object The created object.
object:setGlobalCallback (fct)
Gives the object a global callback. This will remove the ability to use normal callbacks!

Parameters:

  • fct function The function to run when any property of the object changes.
object:hasGlobalCallback ()
Returns if this object has a global callback.

Returns:

    boolean If this object has a global callback.
object:removeGlobalCallback ()
Removes the global callback from the object.

Returns:

    nil
object:extends (className)
Returns if an object extends the given class. Can be given the class name as a string, or a reference to the class itself.

Parameters:

  • className string or table A class's name as a string or a class.

Returns:

    boolean If the object extends the given class.
