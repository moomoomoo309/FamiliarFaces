Methods

camera:new (_, args)
Creates the instance of the camera with the values passed in args.

Parameters:

  • _ (Unused) Allows camera.new to be called or camera:new.
  • args table A table of arguments modifying the camera. Can be any of the following:
    x: The x coordinate of the camera. Defaults to 0.
    y: The y coordinate of the camera. Defaults to 0.
    w: The width of the camera's viewport. Defaults to the screen width.
    h: The height of the camera's viewport. Defaults to the screen height.
    viewport: Another way of specifying the width and height of the camera. Defaults to the screen dimensions.
    zoom: How much the camera should be zoomed in. Defaults to 1.
    rotation: How much the camera should be rotated, in degrees. Defaults to 0.

Returns:

    camera The instance of the camera. Will error if the camera instance already exists.

camera:transition (time, interpolation, values, key, fct)
Transitions the values in values using the interpolation provided in the time provided, running fct. key can be used to make sure it cancels the function with the given key, and fct will run on each iteration.

Parameters:

  • time number How many seconds it should take for the value to change completely.
  • interpolation string (Optional) The key of the function in camera.interpolations to use to tween the values.
  • values table A table whose keys are the fields to transition, and whose values are what they should transition to.
  • key string (Optional) The key of the transition function, so other functions on that key can be cancelled.
  • fct function (Optional) A function to run on each iteration of the transition.

Returns:

    A function which will cancel the transition.
camera:pan (x, y, time, interpolation, fct)
Pans the camera to the given location in the given amount of time using the given interpolation.

Parameters:

  • x number Where it should end up on the x coordinate.
  • y number Where it should end up on the y coordinate.
  • time number How many seconds it should take to pan completely.
  • interpolation string (Optional) The key of the function in camera.interpolations to use to tween the values.
  • fct function A function to run on each movement of the camera.

Returns:

    A function which will cancel the camera pan right where it is.
