Class audioHandler

A class handling the playing, pausing, and looping of audio.

Methods

Methods

audioHandler:add (filePath, fileName)
Adds an audio file into the audio subsystem. The file will override a file with the same name if it has an extension with higher priority, as defined in the extensionPriorities table. It will not load it if they are the same. It can be given a filepath, or an audio object and a name for the audio object.

Parameters:

  • filePath The path to the audio file.
  • fileName The name of the file to add to the audio handler.

Returns:

    nil
audioHandler:remove (fileName)
Removes the audio object with the given name from the audio handler, if it exists.

Parameters:

Returns:

    nil
audioHandler:play (fileName, callback)
Plays the audio object with the given name from the audio handler, if it exists. Returns a function to stop playing the audio file.

Parameters:

  • fileName The name of the file to play.
  • callback (Optional) A callback to run when the file stops playing.

Returns:

    A function which will cancel the playing of this file. If a truthy value is passed and callback is a function, it will also be run.
audioHandler:loop (fileName, callback)
Loops the audio object with the given name from the audio handler, if it exists. Returns a function to stop playing the audio file.

Parameters:

  • fileName The name of the file to play.
  • callback (Optional) A callback to run when the file loops.

Returns:

    A function which will cancel the playing of this file. If a truthy value is passed and callback is a function, it will also be run.
