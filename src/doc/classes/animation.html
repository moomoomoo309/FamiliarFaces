Class animation

A class allowing sprites to be animated.

Fields

animation.animation

Methods

animation:new (args) Creates a new animation with the arguments provided.
animation:start (self) Starts the animation.
animation:stop (self) Stops the animation.
animation:pause (self) Pauses the animation.
animation:resume (self) Resumes the animation.
animation:reset (self) Resets all properties of this animation involving its running state.

animation:animate (self) Animates the sprite, advancing which frame it is on if necessary, or updating the time until the next frame.

animation:new (args)
Creates a new animation with the arguments provided.

Parameters:

  • args A table containing arguments, which may be any of the following: frames: (Optional) The frames of the animation. frameDurations: (Optional) The duration, in seconds, of each frame of the animation, or of all of them. Defaults to 1/60 sprite: The drawable used in the animation colors: (Optional) A table containing all of the colors which will overlay the sprite, or one color which will overlay all of them.

Returns:

    The created animation.
animation:start (self)
Starts the animation. Makes the sprite draw using the first frame of this animation.

Parameters:

  • self The animation to start.

Returns:

    nil
animation:stop (self)
Stops the animation. Makes the sprite go back to drawing using its image property.

Parameters:

  • self The animation to stop.

Returns:

    nil
animation:pause (self)
Pauses the animation. Note, a paused animation will use the current frame of the animation, while a sprite not animating will use the sprite's image.

Parameters:

Returns:
