Class animation

A class allowing sprites to be animated.

Fields

animation.animation

Methods

animation:new (args) Creates a new animation with the arguments provided.
animation:start () Starts the animation.
animation:stop () Stops the animation.
animation:pause () Pauses the animation.
animation:resume () Resumes the animation.
animation:reset () Resets all properties of this animation involving its running state.
animation:copy () Makes a copy of this animation, resetting its state in the process.
animation:animate () Animates the sprite, advancing which frame it is on if necessary, or updating the time until the next frame.
animation:animateAll () Animates all animations at once.

animation:new (args)
Creates a new animation with the arguments provided.

Parameters:

  • args table A table containing arguments, which may be any of the following: frames: (Optional) The frames of the animation. frameDurations: (Optional) The duration, in seconds, of each frame of the animation, or of all of them. Defaults to 1/60 sprite: The drawable used in the animation colors: (Optional) A table containing all of the colors which will overlay the sprite, or one color which will overlay all of them.

Returns:

    animation The created animation.

Stops the animation. Makes the sprite go back to drawing using its image property.

Pauses the animation. Note, a paused animation will use the current frame of the animation, while a sprite not animating will use the sprite's image.
