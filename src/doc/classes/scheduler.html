Class scheduler

A class allowing for functions to be scheduled based on certain conditions.

Fields

scheduler.unpause An alias to scheduler.resume

Methods

scheduler:sleep (seconds) Sleeps the running coroutine until seconds has passed.
scheduler:after (seconds, fct, group) Run fct after seconds has passed.
scheduler:before (seconds, fct, cancelFct, group) Run fct until seconds has passed.
scheduler:_until (conditionFct, fct, cancelFct, group) Runs fct until conditionFct returns a truthy value or the function returned is called.
scheduler:when (conditionFct, fct, cancelFct, group) Runs fct when conditionFct returns a truthy value or the function returned is called.
scheduler:every (seconds, fct, cancelFct, group) Runs fct every seconds seconds.
scheduler:everyCondition (conditionFct, fct, cancelFct, group) Runs fct every time conditionFct returns a truthy value.

scheduler:resume (group) Resumes the given group.
scheduler:update (dt) Updates all of the functions in the scheduler, checking if any need to run and running them if necessary.

Methods

scheduler:sleep (seconds)
Sleeps the running coroutine until seconds has passed. Will not work on the main thread.

Parameters:

  • seconds number How many seconds to sleep for.

Returns:

    nil
scheduler:after (seconds, fct, group)
Run fct after seconds has passed. Returns a function which cancels this function.

Parameters:

  • seconds number How many seconds to wait before running fct.
  • fct function The function to run after the time has passed.
  • group string (Optional) The group the function should be in. Defaults to "default"

Returns:

    function a function which cancels this function.
scheduler:before (seconds, fct, cancelFct, group)
Run fct until seconds has passed. Returns a function which cancels this function.

Parameters:

  • seconds number How many seconds to wait until fct should stop being run.
  • fct function The function to continuously run before the time has passed.
  • cancelFct function/nil (Optional) A function to run if this one is cancelled.
  • group string (Optional) The group the function should be in. Defaults to "default".

Returns:

    function a function which cancels this function.
scheduler:_until (conditionFct, fct, cancelFct, group)
Runs fct until conditionFct returns a truthy value or the function returned is called.

Parameters:

  • conditionFct function A function supplying the condition that will stop fct.
  • fct function The function to run continuously while conditionFct returns a truthy value.
  • cancelFct function (Optional) A function to run if this one is cancelled.
  • group string (Optional) The group the function should be in. Defaults to "default".

Returns:

    function a function which cancels this function.
scheduler:when (conditionFct, fct, cancelFct, group)
Runs fct when conditionFct returns a truthy value or the function returned is called.

Parameters:

  • conditionFct function A function supplying the condition that will run fct.
  • fct function The function to run when conditionFct returns a truthy value.
  • cancelFct function (Optional) A function to run if this one is cancelled.
  • group string (Optional) The group the function should be in. Defaults to "default".

Returns:

    function a function which cancels this function.
scheduler:every (seconds, fct, cancelFct, group)
Runs fct every seconds seconds.

Parameters:

  • seconds number The number of seconds to wait between running fct.
  • fct function The function to run every seconds seconds.
  • cancelFct function (Optional) A function to run if this one is cancelled.
  • group string (Optional) The group the function should be in. Defaults to "default".

Returns:

    function a function which cancels this function.
scheduler:everyCondition (conditionFct, fct, cancelFct, group)
Runs fct every time conditionFct returns a truthy value.

Parameters:

  • conditionFct function A function supplying the condition that will run fct.
  • fct function The function to run every time conditionFct returns a truthy value.
  • cancelFct function (Optional) A function to run if this one is cancelled.
  • group string (Optional) The group the function should be in. Defaults to "default".

Returns:

    string a function which cancels this function.
scheduler:pause (group)
Pause the given group.

Parameters:

  • group any The key of the group to pause. Can technically be any value.

Returns:

    nil
scheduler:resume (group)
Resumes the given group.

Parameters:

  • group any The key of the group to resume. Can technically be any value.

Returns:

    nil

Updates all of the functions in the scheduler, checking if any need to run and running them if necessary.

Parameters:
