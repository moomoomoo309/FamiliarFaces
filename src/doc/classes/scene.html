scene.printText2 (text, paddingX, paddingY, i, color)
The internal function used by printText to write to the screen.

Parameters:

  • text The text to write to the screen.
  • paddingX How many pixels from the left/right side should be left by the text.
  • paddingY How many pixels from the top/bottom side should be left by the text.
  • i Which row the text should be on (1-however many fit on the screen)
  • color (Optional) What color the text should be. Defaults to white.

Methods

scene:printText (_, text, reset, color)
Print the given text on the screen, moving the camera down when the text gets off screen. Make reset true to move the text back to the top of the screen.

Parameters:

  • _ unused
  • text The text to print on screen
  • reset (Optional) If true, will reset the text back to the top of the screen.
  • color (Optional) The color to make the text. White by default.

Returns:

    nil
