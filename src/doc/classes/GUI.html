Class GUI

The class handling all parts of the GUI.

Fields

GUI.update Updates the GUI.
GUI.textinput Passes textinput events to gooi.
GUI.keypressed Passes keypressed events to gooi.
GUI.mousemoved Passes mousemoved events to gooi.

Methods

GUI:addMenu (menu, name) Adds a menu with the passed values and the given name.
GUI:showMenu (menuName) Shows the menu with the given name.
GUI:hideMenu (menuName) Hides the menu with the given name.
GUI:changeMenu (menuName) Changes the visible menu to the one with the given name.
GUI:init () Initializes the GUI, loading any necessary sprites.
GUI:pause () Pauses the game.
GUI:unpause () Unpauses the game.
GUI:paused () Returns if the game is paused or not.
GUI:draw () Draws all GUI-related sprites, and the GUI itself.
GUI:mousepressed () Passes mousepressed events to gooi.
GUI:mousereleased () Passes mousereleased events to gooi.
GUI:startGame () Starts the game.
