process (path)
Processes the file at the given path using require(). Returns a coroutine to the parser and the table it's reading from, or false if it is unsuccessful.

Parameters:

  • path string The path to the file to parse.

Returns:

    coroutine,table or false A coroutine to the parser and the table it's reading from, or false if it is unsuccessful.
